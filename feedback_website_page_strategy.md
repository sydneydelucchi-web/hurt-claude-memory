name: Partner website page strategy — consolidate, don't fragment

type: feedback

Don't recommend partners create additional separate pages (e.g., /digital-rehab, /joint-score, /after-hours) that cause too many clicks away from the poweredbyhurt landing page.

**Why:** Each separate page fragments traffic and dilutes the connection to HURT!. The poweredbyhurt landing page should remain the primary destination. After-Hours is specifically handled by the poweredbyhurt page — no partner web copy kit needed for it.

**How to apply:**
- The poweredbyhurt page handles After-Hours access — do NOT create web copy for an After-Hours page
- Web copy kits are only appropriate for service lines partners deliver themselves (Digital Rehab, Joint Score™ lead capture, general services like Sports Medicine)
- Batch website additions — don't recommend many separate pages
- Recommend sections within existing pages or one consolidated HURT!-powered services page
- The poweredbyhurt landing page is the canonical destination — partner pages should funnel to it, not compete with it
